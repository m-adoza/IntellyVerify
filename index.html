About IntelliVerify

IntelliVerify is an intelligent web platform designed to simplify and secure the final year project topic approval process. It connects students, supervisors, and administrators in a transparent workflow.

With built-in plagiarism detection using TF-IDF and Cosine Similarity, the system helps maintain academic integrity by identifying similar topics before approval.

Developed as a university final year project, IntelliVerify is built with modern technologies and a scalable architecture ready for future AI integration.
